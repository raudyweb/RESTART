# RESTART COFFEE — v1

Web oficial del restaurante RESTART COFFEE, Vilanova i la Geltrú.

Desarrollado por **RAUDY**.

## Tecnologías

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Comandos

```bash
npm install
npm run dev      # desarrollo local
npm run build    # build para producción
```
